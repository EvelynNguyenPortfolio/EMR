# EMR

A commandline software program for managing electronic medical records.

# Environment Configuration

To run the source code, you need to set up your environment variables correctly.
If using VSCode, create a `.env` file in the root directory of the project and add the following MySQL configuration environment variables:

```
DB_URL=jdbc:mysql://localhost:3306/dbname
DB_USER=your_username
DB_PASSWORD=your_password
```
